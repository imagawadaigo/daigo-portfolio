できること

Web制作・資料作成を中心に、「伝える」を形にするお手伝いをしています。
構成設計から文章整理まで、まるごと対応可能です。

主なサービス

Web制作 メイン

飲食店・個人事業主・フリーランスの方向けに、目的に合ったWebページを制作します。
店舗紹介サイト、LP（ランディングページ）、プロフィールサイトなどに対応。ヒアリングで要件を整理し、構成設計→コーディングまで一貫して担当します。

店舗サイト LP プロフィール レスポンシブ SEO設計

資料作成 メイン

営業資料・説明資料・提案書など、ビジネスで使う資料の作成をお手伝いします。
「伝えたい内容はあるけど、うまくまとめられない」という段階からOK。情報の整理と構成を一緒に考えながら仕上げます。

営業資料 説明資料 提案書 PowerPoint PDF

構成設計・文章整理

「何をどの順番で伝えるか」を設計します。Webページのセクション構成、見出しの組み立て、文章のトーン調整など。
サービス説明や自己紹介文の整理もお任せください。言いたいことを聞き取って、読みやすく整えます。

業務改善のご相談

「この作業もっと楽にできないかな？」といったご相談もお気軽にどうぞ。できる範囲でご提案します。

対応範囲について

対応可 こんなご依頼に対応しています

  • HTML / CSS / JavaScript でのWebページ制作
  • WordPress等のテーマを使ったサイト構築
  • LPや店舗サイトの構成設計〜制作
  • PowerPoint・Google Slidesでの資料作成
  • 文章の整理・リライト・トーン調整
  • レスポンシブ対応（スマホ・タブレット）
  • SEOを意識した基本設計

対応外 現在お受けしていないもの

  • 大規模なWebアプリケーション開発
  • ECサイトの構築・決済機能の実装
  • サーバーサイド開発（バックエンド）
  • ロゴ・イラスト等のグラフィックデザイン
  • 動画編集・映像制作
  • 広告運用・SNS運用代行

※対応外の内容でも、ご相談いただければ知り合いをご紹介できる場合があります。

よくある質問

Q.サーバーやドメイン、WordPressなどの「箱」は誰が用意しますか？

ケースバイケースで対応しています。まだ何もない状態ならこちらで用意・提案できますし、すでにサーバーやWordPressをお持ちであればアクセス権をいただいて作業することも可能です。ヒアリング時に一緒に決めましょう。

Q.納品はどんな形式になりますか？

主に3つのパターンがあります。①HTMLファイル一式をお渡し、②GitHub Pagesでの公開、③WordPressへの構築です。ご希望やご予算に合わせてご提案しますので、事前に決まっていなくても大丈夫です。

Q.写真や文章がなくても依頼できますか？

はい、可能です。フリー素材の提案や、文章の整理・ライティングもお手伝いできます。「素材がないから頼めない」と思わず、お気軽にご相談ください。

Q.納品後、自分で更新できますか？

WordPressで構築した場合は、管理画面から文章や画像の差し替えが可能です。HTMLファイルでの納品の場合は基本的にコード編集が必要ですが、更新のしやすさを重視される場合はCMS導入をご提案します。

Q.スマホ対応はしてもらえますか？

はい、全案件でレスポンシブ対応（スマホ・タブレット表示の最適化）を標準で行っています。追加費用はかかりません。

Q.完成までどのくらいかかりますか？

内容やページ数にもよりますが、LP1ページで1〜2週間が目安です。お急ぎの場合はスケジュールのご相談も可能です。詳しくは制作の流れもご覧ください。
