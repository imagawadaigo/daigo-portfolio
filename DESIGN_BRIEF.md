## Claude Design への投げ方

以下のプロンプトをこのブリーフと一緒に貼り付ける。

---

**【プロンプト】**

> このブリーフを元に、フリーランス開発者のポートフォリオサイトをリニューアルしてほしい。
>
> まず **index.html（トップページ）** を HTML + CSS で一から作り直してほしい。現行サイトのコードは参考程度で、デザインは完全に新しく作って構わない。
>
> 優先してほしいのは以下の3点：
> 1. 「人」が見えること。写真と一言で温かみのある人間が伝わる Hero
> 2. 「この人は本気でプロダクトを作る人だ」という信頼感。実績（本番稼働SaaS・EC開発中）を視覚的に際立たせる
> 3. デザイン自体が「センスのある人が作った」と感じさせること
>
> 参考サイトのビジュアル言語（余白・タイポグラフィ・ミニマルさ）を取り入れつつ、日本語テキストが自然に馴染む形にしてほしい。
> 技術的制約：静的HTML + CSS のみ（Jekyll / GitHub Pages）。JavaScriptは最小限でよい。

## 現行サイトの問題点

- 技術スタック名の羅列になっていて「人」が見えない
- 若いフリーランスが多い中で埋もれる。個性が出ていない
- デザインが平凡で、訪問者が「センスのある人」と判断できない
- 文章の課題解決軸は良いが、視覚的な印象付けが弱い

---

## 目指したいトーン・印象

- 温かみがあるが、チープではない
- 清潔感・余白・読みやすさ。華美でなくていい
- 「プロが偶然まだ若い」という雰囲気
- 参考トーン：Linear・Notion のような洗練されたプロダクトサイト。個人開発者ポートフォリオ的な誠実さ
